# surface-of-revolution-with-Bezier-curves-opengl
## 实现效果
将用户交互式输入的图形用Bezier曲线平滑。
![image]{https://github.com/skylancer2019/surface-of-revolution-with-Bezier-curves-opengl-/tree/main/pic/rawpic.png}
